MCDS

MCDS is a Cytoscape app that allows users to find an approximate MCDS solution in a directed graph. The user can choose to run the algorithm on the largest connected component underlying directed graph (LCCD) and undirected graph (LCC), as well as the strongly connected component (SCC) of the network. The connectivity among the nodes in the MCDS can then be visualized via Cytoscape.
