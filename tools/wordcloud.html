WordCloud

WordCloud is a Cytoscape app that generates a word tag cloud from a user-defined node selection, summarizing an attribute of choice. For instance, if selected nodes are proteins, and the string attribute "full protein name" is selected, every string will be broken down into words, which will be plotted on a panel with size proportional to their frequency. It is also possible to use the app to cluster words that appear together in the selected nodes.
