PiHelper

PiHelper is a drug- and antibody-target information aggregator and provider service. The data aggregated by PiHelper can be accessed either programmatically via a Java API or through a web user interface. The framework works in a human-gene centric manner and model drug-target and antibody-target relationships accordingly.
